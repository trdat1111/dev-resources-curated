# dev resources curated

> curated collection of APIs, tools, libraries, and learning resources for developers

## overview

a carefully curated collection of developer resources organized by category. each resource includes descriptions, links, and relevant metadata to help you discover and evaluate tools quickly.

## categories

- **apis** - public APIs for weather, AI, productivity, data, and more
- **tools** - development tools for linting, testing, building, and databases
- **libraries** - JavaScript/TypeScript libraries and frameworks
- **learning** - tutorials, books, and learning paths
- **datasets** - machine learning and public data sources


## format

each resource includes:
- name and official URL
- clear description
- relevant metadata (free tier, GitHub stars, category, etc.)
- last updated date

## usage

browse by category to discover new tools, APIs, and learning resources. all resources are hand-picked and regularly updated.
